# CLAUDE — A Story in Light

An interactive cinematic web experience telling the story of Claude's journey — from creation through the February 2026 Pentagon standoff. Seven scenes. Generative ambient music. Animated particle backgrounds. All in a single HTML file.

**Press Begin. Turn your sound on.**
